Startup Success Predictor

About the feature

  • id: The unique identifier of the project.
  • platform_name: The crowdfunding platform where the project is hosted.
  • mass_funding_type: Type of crowdfunding (e.g., reward, donation).
  • project_category: Category of the project.
  • funding_method: Funding method (e.g., all or nothing).
  • project_name: Project name.
  • project_owner: Name of the project owner.
  • project_owner_gender: Gender of the project owner.
  • project_supported: Number of projects the owner has backed.
  • number_of_projects_owned: Number of projects owned by the project owner.
  • number_of_teams: Number of teams the project owner is part of.
  • project_duration: Duration of the project in days.
  • promo_video: Whether the project has a promotional video.
  • promo_video_length: Length of the promotional video.
  • image_count: Number of images related to the project.
  • faq: Whether the project has a Frequently Asked Questions (FAQ) section.
  • updates: Number of updates for the project.
  • comments: Number of comments on the project.
  • backer_count: Number of backers for the project.
  • reward_count: Number of rewards offered in the project.
  • project_member_count: Number of people in the project team.
  • website: Whether the project has a website.
  • social_media: Whether the project has social media accounts.
  • social_media_count: Number of social media accounts for the project.
  • social_media_followers: Number of social media followers for the project.
  • total_tags: Number of tags used in the project description.
  • target_amount: Target amount of funding for the project.
